# Lighting Calculations

Solar intensity and lighting time analysis.

## Solar Intensity

Compute solar intensity on spacecraft or ground sites.

- **Spacecraft Example**: `examples/07_lighting/solar_intensity.py`
- **Site Example**: `examples/07_lighting/solar_intensity.py`

## Lighting Times

Calculate eclipse and sunlit periods for spacecraft.

- **Example**: `examples/07_lighting/lighting_times.py`

## Solar AER

Solar position relative to ground sites (azimuth, elevation, range).

- **Example**: `examples/07_lighting/solar_aer.py`

## Applications

- Power budget planning
- Thermal analysis
- Optical payload scheduling
- Eclipse predictions
