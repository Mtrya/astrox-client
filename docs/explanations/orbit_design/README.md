# Orbit Design

Utilities for designing common orbit types and constellations.

## Standard Orbit Types

### Geostationary Orbit (GEO)

Stationary orbit for communications satellites.

- **Example**: `examples/04_orbit_design/geo_orbit.py`
- **Theory**: [Geostationary orbit (Wikipedia)](https://en.wikipedia.org/wiki/Geostationary_orbit)

### Molniya Orbit

Highly elliptical orbit for high-latitude coverage.

- **Example**: `examples/04_orbit_design/molniya_orbit.py`
- **Theory**: [Molniya orbit (Wikipedia)](https://en.wikipedia.org/wiki/Molniya_orbit)

### Sun-Synchronous Orbit (SSO)

Polar orbit with constant sun angle.

- **Example**: `examples/04_orbit_design/sso_orbit.py`
- **Theory**: [Sun-synchronous orbit (Wikipedia)](https://en.wikipedia.org/wiki/Sun-synchronous_orbit)

## Constellations

### Walker Constellation

Symmetric satellite constellations.

- **Example**: `examples/04_orbit_design/walker_constellation.py`
- **Theory**: [Walker constellation (Wikipedia)](https://en.wikipedia.org/wiki/Walker_constellation)

## Coordinate Conversions

Convert between orbital element representations.

- **Example**: `examples/04_orbit_design/coordinate_conversions.py`
