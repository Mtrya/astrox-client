# Rocket Trajectory

Rocket ascent and descent trajectory optimization.

## Ascent Optimization

Optimize rocket ascent trajectory for payload delivery.

- **Example**: `examples/06_rocket/ascent_optimization.py`

## Descent Optimization

Optimize rocket descent and landing trajectory.

- **Example**: `examples/06_rocket/descent_optimization.py`

## Guidance Trajectory

Calculate guidance trajectories for launch vehicles.

- **Example**: `examples/06_rocket/guidance_trajectory.py`

## Chinese Launch Vehicles

The ASTROX API includes guidance models for Chinese launch vehicles:

- CZ-2C/D
- CZ-3B/C
- CZ-4B/C
- CZ-7A
- KZ-1A

See individual example files for vehicle-specific guidance patterns.
